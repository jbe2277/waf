Readme
------

This project contains C# code snippets for developing applications based on the WPF Application Framework (WAF).
All code snippets start with "waf".

   
Activate the code snippets:

1) Start Visual Studio 2012
2) Click on "Tools" and then on "Code Snippets Manager..."
3) Click on "Add" and choose the "WAF Snippets" directory of this project.
